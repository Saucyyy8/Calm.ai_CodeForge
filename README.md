## ğŸš€ Getting Started

### Prerequisites
*   Node.js & npm
*   Java JDK 17+
*   Maven

### Backend Setup
1.  Navigate to the project root.
2.  Run the Spring Boot application:
    ```bash
    ./mvnw spring-boot:run
    ```

### Frontend Setup
1.  Navigate to the frontend directory:
    ```bash
    cd frontend
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
